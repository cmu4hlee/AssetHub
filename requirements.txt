# ============================================
# 1. 安装后端依赖:
#    cd backend && npm install
#
# 2. 安装前端依赖:
#    cd frontend && npm install
#
# 3. 或使用启动脚本自动安装:
#    ./start.sh (Linux/Mac)
#    start.bat (Windows)

# ============================================
# 环境变量配置
# ============================================
# 后端需要配置 .env 文件，参考 backend/.env.example
# 主要配置项:
# - DB_HOST: 数据库主机
# - DB_PORT: 数据库端口
# - DB_USER: 数据库用户名
# - DB_PASSWORD: 数据库密码
# - DB_NAME: 数据库名称
# - JWT_SECRET: JWT密钥
# - PORT: 服务器端口 (默认: 3000)
# - DASHSCOPE_API_KEY 或 OPENAI_API_KEY: AI服务密钥 (可选)
