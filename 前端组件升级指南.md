## 一、升级概览

### 1.1 升级目标

1. ✅ 利用已有的幂等性机制
2. ✅ 简化错误处理
3. ✅ 统一操作模式
4. ✅ 提升用户体验

### 1.2 当前状态

**好消息：** API客户端（`frontend/src/api/client.js`）**已经自动处理幂等性**！

**当前实现：**
```javascript
// API客户端已自动添加幂等性键
api.interceptors.request.use(config => {
  ensureIdempotencyKey(config); // ✅ 自动添加
  return config;
});

// 响应拦截器自动处理高风险确认
api.interceptors.response.use(response => {
  return response.data;
}, async error => {
  if (payload?.requiresConfirmation) {
    // ✅ 自动处理高风险确认
    originalRequest.headers['X-Risk-Confirm-Token'] = payload.confirmToken;
    return api.request(originalRequest);
  }
});
```

**这意味着：** 现有组件使用 `maintenanceAPI.createMaintenanceLog()` **已经可以正常工作了**！

```javascript
if (isEditing) {
```

```javascript
import { createMaintenanceLog } from '../utils/highRiskOperation';

let response;
if (isEditing) {
```

### 3.3 升级方案二：使用React Hook（完整升级）

**升级后的代码：**
```javascript
import { useHighRiskOperation } from '../hooks/useHighRiskOperation';

function MaintenanceLogForm({ record, isEditing, onSubmit, onCancel }) {
  const { createLog, loading } = useHighRiskOperation({
    onSuccess: (data) => {
      message.success('创建成功');
      onSubmit();
    },
    onError: (error) => {
      message.error('创建失败: ' + error.message);
    }
  });

  const handleSubmit = async () => {
    try {
      const values = await form.validateFields();
      // 格式化数据...
      const formData = { ... };

      if (isEditing) {
        // 编辑模式使用原有API
        await maintenanceAPI.updateMaintenanceLog(record.id, formData);
      } else {
        // 创建模式使用新Hook
        await createLog(formData);
      }
    } catch (error) {
      // 错误处理...
    }
  };

  return (
    <Spin spinning={loading}>
      <Form form={form} onFinish={handleSubmit}>
        {/* 表单内容 */}
      </Form>
    </Spin>
  );
}
```

**优势：**
- ✅ 更好的状态管理
- ✅ 自动loading状态
- ✅ 统一的错误处理

**文件：** `frontend/src/pages/MaintenanceRequestForm.jsx`

```javascript
import { maintenanceAPI } from '../utils/api';

const handleSubmit = async (values) => {
  setLoading(true);
  try {
    const response = await maintenanceAPI.createMaintenanceRequest(values);
    if (response.success) {
      message.success('创建成功');
      onSuccess();
    } else {
      message.error(response.message || '创建失败');

  } catch (error) {

  } finally {
    setLoading(false);
  }
};
```

**升级后：**
```javascript
import { createMaintenanceRequest } from '../utils/highRiskOperation';

const handleSubmit = async (values) => {
  setLoading(true);
  try {
    const result = await createMaintenanceRequest(values, {
      showConfirm: true,
      confirmTitle: '确认提交',
      confirmContent: '确定要提交这个维修申请吗？',
    });

    if (result.success) {
      message.success('创建成功');
      onSuccess();
    } else if (result.cancelled) {
      // 用户取消
    } else {
      message.error(result.error || '创建失败');
    }
  } catch (error) {
    message.error('提交失败: ' + error.message);
  } finally {
    setLoading(false);
  }
};
```

## 十、最佳实践

### 10.1 代码规范

**✅ 推荐：**
```javascript
import { createMaintenanceLog } from '../utils/highRiskOperation';

const result = await createMaintenanceLog(formData, {
  showConfirm: true,
  confirmTitle: '确认创建',
  confirmContent: '确定要创建这条记录吗？',
  onSuccess: (data) => {
    message.success('创建成功');
  },
  onError: (error) => {
    message.error(error.message);
  }
});
```

**❌ 避免：**
```javascript
// 不要混用新旧方式
const response = await maintenanceAPI.createMaintenanceLog(formData);
if (response.success) {
  message.success('成功');
}
```

### 10.2 错误处理

**✅ 推荐：**
```javascript
const result = await createMaintenanceLog(data);

if (result.success) {
  // 成功处理
  message.success('创建成功');
} else if (result.cancelled) {
  // 用户取消
} else {
  // 错误处理
  message.error(result.error);
}
```

**❌ 避免：**
```javascript
// 不要忽略错误
await createMaintenanceLog(data);
message.success('完成');
```

### 10.3 Loading状态

**✅ 推荐：**
```javascript
const { createLog, loading } = useHighRiskOperation();

return <Button loading={loading}>提交</Button>;
```

**❌ 避免：**
```javascript
// 不要手动管理loading
const [loading, setLoading] = useState(false);
const handleSubmit = async () => {
  setLoading(true);
  await createLog(data);
  setLoading(false);
};
```
