# 资产标签系统配置与优化报告

## 1. 系统现状分析

### 1.1 问题诊断
- **API错误**: 前端控制台显示API错误和500 Internal Server Error
- **标签打印问题**: 资产标签打印功能可能存在关联字段不一致的问题
- **数据库关联**: 部分表使用asset_id，部分表使用asset_code，导致关联不一致

### 1.2 系统架构
- **前端**: React + Ant Design，使用AssetLabelDesigner组件设计标签
- **后端**: Node.js + Express，提供资产标签相关API
- **数据库**: MySQL，包含asset_label_templates和asset_label_print_queue表

## 2. 数据库结构优化

- **打印队列**: 优化打印任务管理，使用asset_code关联资产

### 3.2 前端组件优化

- **字段映射**: 支持将标签元素映射到资产字段

## 6. 技术文档

### 6.1 API接口
- **GET /api/asset-labels/templates**: 获取标签模板列表
- **POST /api/asset-labels/templates**: 创建标签模板
- **GET /api/asset-labels/generate-zpl/{templateId}/{assetCode}**: 生成ZPL标签
- **POST /api/asset-labels/print**: 打印标签
- **GET /api/asset-labels/print-queue**: 获取打印队列

### 6.2 前端组件
- **AssetLabelDesigner**: 标签设计器组件
- **标签元素类型**: text, barcode, qr_code, line, rectangle, circle
- **资产字段映射**: asset_code, asset_name, category, department_name等

## 9. 技术支持

### 9.1 常见问题
- **标签预览空白**：检查模板元素配置，确保至少包含一个元素
- **打印失败**：检查打印机IP和端口设置，确保网络连接正常
- **ZPL生成错误**：检查资产编码是否存在，模板配置是否正确

### 9.2 故障排除
- **数据库连接错误**：检查数据库配置和网络连接
- **API 500错误**：检查服务器日志，确认数据库表结构是否正确
- **标签打印偏移**：调整标签尺寸和打印机设置
