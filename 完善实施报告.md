### 2.2 智能预警系统实现

#### 后端实现

1. **智能预警服务** (`backend/services/intelligent-alert-service.js`)
   - 预警概览统计
   - 保养到期预警检查
   - 资质到期预警检查
   - 特种设备检验到期预警
   - 安全检测到期预警
   - 开机率异常预警
   - 预警设置管理

2. **预警路由** (`backend/routes/intelligent-alerts.js`)
   - `GET /api/intelligent-alerts/overview` - 预警概览
   - `GET /api/intelligent-alerts` - 预警列表
   - `GET /api/intelligent-alerts/maintenance` - 保养预警
   - `GET /api/intelligent-alerts/qualifications` - 资质预警
   - `GET /api/intelligent-alerts/inspections` - 检验预警
   - `GET /api/intelligent-alerts/safety` - 安全检测预警
   - `GET /api/intelligent-alerts/uptime` - 开机率预警
   - `GET /api/intelligent-alerts/settings` - 获取设置
   - `POST /api/intelligent-alerts/settings` - 保存设置

3. **数据库表** (`backend/scripts/create-alert-tables.sql`)
   - `alert_settings` - 用户预警设置表
   - `alert_records` - 预警记录表
   - `alert_notification_logs` - 预警通知日志表

4. **服务器路由注册** (`backend/server.js`)
   - 添加 `/api/intelligent-alerts` 路由

#### 前端实现

1. **API接口** (`frontend/src/utils/api.js`)
   ```javascript
   export const intelligentAlertAPI = {
     getOverview: () => api.get('/intelligent-alerts/overview'),
     getAlerts: params => api.get('/intelligent-alerts', { params }),
     getMaintenanceAlerts: days => api.get('/intelligent-alerts/maintenance', { params: { days } }),
     getQualificationAlerts: days => api.get('/intelligent-alerts/qualifications', { params: { days } }),
     getInspectionAlerts: days => api.get('/intelligent-alerts/inspections', { params: { days } }),
     getSafetyAlerts: days => api.get('/intelligent-alerts/safety', { params: { days } }),
     getUptimeAlerts: threshold => api.get('/intelligent-alerts/uptime', { params: { threshold } }),
     getSettings: () => api.get('/intelligent-alerts/settings'),
     saveSettings: data => api.post('/intelligent-alerts/settings', data),
   };
   ```

2. **预警通知组件** (`frontend/src/components/AlertNotification.jsx`)
   - 顶部导航栏通知图标
   - 下拉展示预警列表
   - 支持按类型筛选
   - 紧急预警红色提醒
   - 每5分钟自动刷新

3. **预警中心页面** (`frontend/src/pages/AlertCenter.jsx`)
   - 预警统计卡片
   - 预警列表表格
   - 类型和紧急程度筛选
   - 快速跳转到相关页面

4. **布局集成** (`frontend/src/components/Layout.jsx`)
   - 在 Header 中添加 AlertNotification 组件

5. **路由配置** (`frontend/src/App.jsx`)
   - 添加 `/alert-center` 路由
