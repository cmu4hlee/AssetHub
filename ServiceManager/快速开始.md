# 🚀 服务管理器 - 快速开始

## 📱 macOS 用户

### 方式一：直接运行（开发模式）
**双击** `启动macOS应用.sh` 即可启动应用

### 方式二：构建独立应用
1. **双击** `构建macOS应用.sh` 构建应用
2. 构建完成后，在 `dist` 目录找到 `.dmg` 文件
3. 双击 `.dmg` 安装应用
4. 在启动台中找到"资产管理服务管理器"运行

---

## 🪟 Windows 用户

### 方式一：直接运行（开发模式）
**双击** `启动Windows应用.bat` 即可启动应用
- 如果已编译，直接运行
- 如果未编译，会自动编译后运行

### 方式二：构建独立应用
1. **双击** `构建Windows应用.bat` 构建应用
2. 构建完成后，在 `publish\win-x64` 目录找到 `ServiceManager.exe`
3. 双击 `ServiceManager.exe` 运行
4. 可以将此文件复制到任何位置使用

---

## ⚙️ 首次使用

1. **启动应用**后，会自动读取项目配置
2. 在"配置管理"标签页中：
   - 检查服务端口配置
   - 检查数据库配置
   - 点击"测试数据库连接"验证配置
3. 点击"保存配置"保存设置
4. 在"服务控制"标签页中：
   - 点击"启动后端服务"
   - 点击"启动前端服务"
   - 或使用"启动所有服务"一键启动

---

## 💡 提示

- **配置文件**：应用会在运行目录创建 `config.json` 保存配置
- **自动检测**：应用会自动查找项目根目录（包含 backend 和 frontend 的目录）
- **自动重启**：可以启用"自动监控和重启"功能，服务异常时自动重启
