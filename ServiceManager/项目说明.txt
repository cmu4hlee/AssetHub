资产管理系统服务管理器
====================

这是一个 Windows Forms 应用程序，用于管理资产管理系统的前端和后端服务。

主要功能：
----------
1. 启动、停止、重启前端和后端服务
2. 配置前端 IP 和端口号
3. 配置后端 IP 和端口号
4. 配置数据库连接信息（IP、端口、数据库名、用户名、密码）
5. 自动保存和加载配置
6. 实时显示服务运行状态

快速开始：
----------
1. 确保已安装 .NET 6.0 或更高版本
2. 双击"编译服务管理器.bat"进行编译
3. 编译成功后，运行生成的 ServiceManager.exe
4. 或在 ServiceManager 目录下运行：dotnet run

配置文件位置：
-------------
%APPDATA%\AssetManagement\config.json

注意事项：
----------
- 确保已安装 Node.js 和 npm
- 确保前端和后端目录已安装依赖（npm install）
- 修改配置后需要重启服务才能生效
- 关闭应用程序时会自动停止所有运行中的服务

文件说明：
----------
- ServiceManager.csproj: 项目文件
- Program.cs: 程序入口
- MainForm.cs: 主窗口界面
- ServiceController.cs: 服务控制逻辑
- AppConfig.cs: 配置管理
- README.md: 详细说明文档
- 使用说明.md: 中文使用说明
