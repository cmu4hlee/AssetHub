# macOS 安装程序构建说明

## 📦 安装程序格式

本服务管理器支持两种 macOS 安装格式：

### 1. .pkg 安装程序（推荐）
- **特点**：标准的 macOS 安装包格式
- **优点**：专业的安装体验，可以安装到系统目录
- **适用场景**：正式发布、批量部署

### 2. .dmg 磁盘映像
- **特点**：磁盘映像文件，拖拽安装
- **优点**：简单快速，无需安装向导
- **适用场景**：快速分发、个人使用

---

## 🚀 快速构建

### 构建 .pkg 安装程序

**双击运行** `构建macOS安装程序.sh`

或者使用命令行：
```bash
cd ServiceManager
npm run build:pkg
```

### 构建 .dmg 磁盘映像

```bash
npm run build:dmg
```

### 构建所有格式

**双击运行** `构建所有macOS格式.sh`

或者使用命令行：
```bash
npm run build:all
```

---

## 📋 构建步骤详解

### 1. 准备环境

确保已安装：
- Node.js (v16 或更高版本)
- npm

### 2. 安装依赖

首次构建需要安装依赖：
```bash
cd ServiceManager
npm install
```

### 3. 构建安装程序

选择构建方式：

**方式一：使用脚本（推荐）**
```bash
# 构建 .pkg 安装程序
./构建macOS安装程序.sh

# 或构建所有格式
./构建所有macOS格式.sh
```

**方式二：使用 npm 命令**
```bash
# 构建 .pkg
npm run build:pkg

# 构建 .dmg
npm run build:dmg

# 构建所有格式
npm run build:all
```

### 4. 查找生成的文件

构建完成后，在 `dist` 目录找到：
- `资产管理服务管理器-1.0.0.pkg` - 安装程序
- `资产管理服务管理器-1.0.0.dmg` - 磁盘映像

---

## 💡 使用安装程序

### .pkg 安装程序使用步骤

1. **双击 .pkg 文件**
   - 系统会打开安装向导

2. **按照向导安装**
   - 点击"继续"
   - 选择安装位置（默认：/Applications）
   - 输入管理员密码
   - 等待安装完成

3. **启动应用**
   - 在启动台找到"资产管理服务管理器"
   - 或在应用程序文件夹中双击运行

4. **首次运行权限设置**
   - 如果遇到安全提示，前往：
     - 系统偏好设置 > 安全性与隐私 > 通用
     - 点击"仍要打开"

### .dmg 磁盘映像使用步骤

1. **双击 .dmg 文件**
   - 系统会挂载磁盘映像

2. **拖拽安装**
   - 将应用图标拖拽到"应用程序"文件夹
   - 或直接双击运行（不安装）

3. **弹出磁盘**
   - 安装完成后，在 Finder 侧边栏点击弹出

## ⚙️ 安装程序配置

### 安装位置

- **默认位置**：`/Applications/资产管理服务管理器.app`
- **可自定义**：安装向导中可选择

### 权限要求

安装程序需要以下权限：
- **管理员权限**：安装到系统目录
- **网络权限**：连接数据库和服务
- **文件访问权限**：读取项目配置文件

### 系统要求

- macOS 10.13 (High Sierra) 或更高版本
- 支持 Intel (x64) 和 Apple Silicon (arm64)

---

## 🔧 高级配置

### 自定义安装程序

编辑 `package.json` 中的 `build.mac` 配置：

```json
{
  "build": {
    "mac": {
      "target": [
        {
          "target": "pkg",
          "arch": ["x64", "arm64"]
        }
      ],
      "category": "public.app-category.utilities"
    }
  }
}
```

### 签名和公证（可选）

如果需要代码签名和公证：

1. **获取开发者证书**
   - 在 Apple Developer 注册账号
   - 下载开发者证书

2. **配置签名**
   ```json
   {
     "build": {
       "mac": {
         "identity": "Developer ID Application: Your Name"
       }
     }
   }
   ```

3. **设置环境变量**
   ```bash
   export APPLE_ID="your@email.com"
   export APPLE_APP_SPECIFIC_PASSWORD="your-password"
   ```

## ⚠️ 常见问题

### 1. 构建失败

**问题**：`electron-builder` 未找到
**解决**：
```bash
npm install --save-dev electron-builder
```

### 2. 权限被拒绝

**问题**：无法写入 dist 目录
**解决**：
```bash
chmod 755 dist
```

### 3. 安装后无法运行

**问题**：安全提示阻止运行
**解决**：
- 系统偏好设置 > 安全性与隐私 > 通用
- 点击"仍要打开"
- 或在终端运行：`xattr -cr /Applications/资产管理服务管理器.app`

### 4. 图标不显示

**问题**：应用图标显示为默认图标
**解决**：
- 确保 `icon.icns` 文件存在
- 图标文件格式正确（.icns）
