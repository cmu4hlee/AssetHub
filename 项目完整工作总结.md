## 📝 最佳实践

### 1. 开发规范

**✅ 推荐：**
```javascript
import { createMaintenanceLog } from '../utils/highRiskOperation';
import { useHighRiskOperation } from '../hooks/useHighRiskOperation';

// 使用Hook
const { createLog, loading } = useHighRiskOperation({
  onSuccess: () => message.success('成功'),
  onError: (error) => message.error(error.message),
});

// 一行代码创建
await createLog({ asset_code: '000000555' });
```

**❌ 避免：**
```javascript
// 不要混用新旧方式
const response = await maintenanceAPI.createMaintenanceLog(data);
if (response.success) {
  message.success('成功');
}
```

### 2. 错误处理

**✅ 推荐：**
```javascript
const result = await createMaintenanceLog(data);

if (result.success) {
  message.success('创建成功');
} else if (result.cancelled) {
  // 用户取消
} else {
  message.error(result.error);
}
```

### 3. 性能优化

**✅ 推荐：**
- 使用 `Promise.all()` 处理批量操作
- 使用防抖处理表单提交
- 合理使用缓存机制

**❌ 避免：**
- 循环中频繁调用创建操作
- 不必要的重复请求
- 大数据量同步处理
