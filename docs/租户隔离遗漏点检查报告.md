## 六、数据库表检查脚本

建议运行以下脚本检查所有表是否有 `tenant_id` 字段：

```sql
SELECT 
  TABLE_NAME,
  CASE 
    WHEN EXISTS (
      SELECT 1 FROM INFORMATION_SCHEMA.COLUMNS 
      WHERE TABLE_SCHEMA = DATABASE() 
      AND TABLE_NAME = t.TABLE_NAME 
      AND COLUMN_NAME = 'tenant_id'
    ) THEN '有'
    ELSE '无'
  END as has_tenant_id
FROM INFORMATION_SCHEMA.TABLES t
WHERE TABLE_SCHEMA = DATABASE()
AND TABLE_TYPE = 'BASE TABLE'
AND TABLE_NAME NOT IN ('tenants', 'users', 'departments', 'roles', 'permission_definitions', 'role_menu_permissions', 'menu_definitions')
ORDER BY TABLE_NAME;
```
