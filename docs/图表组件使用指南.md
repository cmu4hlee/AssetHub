# 图表组件使用指南

## 概述

本项目提供了多种 ECharts 图表组件，用于数据可视化展示。

## 图表组件列表

### 1. 开机率趋势图 (UptimeTrendChart)

展示设备开机率的历史趋势。

```jsx
import { UptimeTrendChart } from '../components/charts';

<UptimeTrendChart 
  data={[
    { month: '2026-01', uptime_rate: 96.5 },
    { month: '2026-02', uptime_rate: 94.2 },
    { month: '2026-03', uptime_rate: 97.8 },
  ]} 
  height={300}
/>
```

### 2. 故障分析图表

#### 故障类型饼图 (FailureTypePieChart)

```jsx
import { FailureTypePieChart } from '../components/charts';

<FailureTypePieChart

    { name: '电气故障', value: 28 },
    { name: '软件故障', value: 15 },

  height={300}
/>
```

### 3. 保养完成率图表

#### 仪表盘 (MaintenanceGaugeChart)

## 数据格式说明

### 开机率数据

```typescript
interface UptimeData {
  month: string;        // 月份，如 "2026-01"
  uptime_rate: number;  // 开机率，0-100
}
```

### 故障数据

```typescript
interface FailureData {
  type?: string;        // 故障类型
  name?: string;        // 名称
  value?: number;       // 数值
  count?: number;       // 次数
  avgTime?: number;     // 平均处理时间（小时）
}
```

### 保养数据

```typescript
interface MaintenanceData {
  month?: string;           // 月份
  department?: string;      // 部门名称
  type?: string;            // 保养类型
  planned?: number;         // 计划数
  completed?: number;       // 完成数
  pending?: number;         // 待执行数
  overdue?: number;         // 逾期数
  completion_rate?: number; // 完成率
  total?: number;           // 总数
}
```

## 样式定制

所有图表组件都支持通过 `height` 属性设置高度，默认为 300px。

图表会自动适应容器宽度，并在窗口大小变化时自动重绘。

## 注意事项

1. 确保已安装 echarts 依赖：`npm install echarts`
2. 图表组件在卸载时会自动销毁实例，避免内存泄漏
3. 数据更新时会自动重新渲染图表
