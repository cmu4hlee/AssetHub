## 🚀 验证方法

### 检查配置是否正确
```bash
# 检查前端配置
cd frontend
cat vite.config.js | grep -A 2 "development"

# 检查后端配置
cd ../backend
cat config/app.config.js | grep -A 2 "isDevelopment"
```

### 启动服务验证
```bash
# 启动后端（应该显示端口6001）
cd backend
npm start

# 启动前端（应该显示端口6000）
cd ../frontend
npm run dev
```

### 检查端口占用
```bash
# 检查开发模式端口
lsof -i:6000
lsof -i:6001

# 检查生产模式端口
lsof -i:4000
lsof -i:4001
```
