# 质量管理模块完善说明

**更新日期**: 2026-01-14  
**模块名称**: 质量管理（原质量控制）

---

## 📋 更新内容

### 1. 模块重命名
- ✅ 将"质量控制"统一改为"质量管理"
- ✅ 更新所有菜单、文档、注释中的名称

### 2. 新增功能

#### 2.1 周期管理和预警功能
- ✅ **计量到期预警**：`GET /api/quality-control/metrology/expiring`
  - 支持自定义预警天数（默认30天）
  - 自动计算剩余天数
  - 按到期日期排序
  
- ✅ **质控到期预警**：`GET /api/quality-control/quality-control/expiring`
  - 支持自定义预警天数（默认30天）
  - 自动计算剩余天数
  - 按到期日期排序

#### 2.2 统计分析功能
- ✅ **计量统计分析**：`GET /api/quality-control/metrology/statistics`
  - 总数统计
  - 按类型统计
  - 按结果统计
  - 按状态统计
  - 按月统计（最近12个月）
  - 总费用统计
  - 即将到期数量统计

- ✅ **质控统计分析**：`GET /api/quality-control/quality-control/statistics`
  - 总数统计
  - 按类型统计
  - 按结果统计
  - 按状态统计
  - 按月统计（最近12个月）
  - 合格率统计
  - 即将到期数量统计

#### 2.3 资产质量管理历史
- ✅ **获取资产质量管理历史**：`GET /api/quality-control/asset/:assetId/history`
  - 合并显示计量和质控历史记录
  - 按日期倒序排列
  - 支持在资产详情页面快速查看

### 3. 数据库表增强

#### 3.1 计量管理表增强字段
- `metrology_cycle` - 计量周期（月）
- `warning_days` - 预警天数（默认30天）
- `certificate_validity_date` - 证书有效期
- `is_auto_remind` - 是否自动提醒

#### 3.2 质控管理表增强字段
- `qc_cycle` - 质控周期（月）
- `warning_days` - 预警天数（默认30天）
- `is_auto_remind` - 是否自动提醒

#### 3.3 新增表
- `quality_management_alerts` - 质量管理预警记录表
  - 记录所有预警信息
  - 支持预警级别（normal/warning/urgent）
  - 支持已读/已处理状态
  
- `quality_management_cycles` - 质量管理周期配置表
  - 资产级别的周期配置
  - 支持计量和质控分别配置周期
  - 支持自定义预警天数

## 🚀 使用说明

### 1. 初始化数据库增强

运行以下脚本增强数据库表结构：

```bash
cd Asset/backend
node scripts/enhance-quality-management-tables.js
```

**注意**：需要数据库服务正在运行

### 2. 使用预警功能

#### 获取即将到期的计量记录
```javascript
// 获取30天内到期的计量记录
const expiring = await qualityControlAPI.getExpiringMetrology({ days: 30 });
```

#### 获取即将到期的质控记录
```javascript
// 获取30天内到期的质控记录
const expiring = await qualityControlAPI.getExpiringQualityControl({ days: 30 });
```

### 3. 使用统计分析功能

#### 获取计量统计
```javascript
const stats = await qualityControlAPI.getMetrologyStatistics({
  start_date: '2024-01-01',
  end_date: '2024-12-31',
  department_id: 1  // 可选
});
```

#### 获取质控统计
```javascript
const stats = await qualityControlAPI.getQualityControlStatistics({
  start_date: '2024-01-01',
  end_date: '2024-12-31',
  department_id: 1  // 可选
});
```

### 4. 查看资产质量管理历史

```javascript
const history = await qualityControlAPI.getAssetQualityHistory(assetId);
// 返回该资产的所有计量和质控历史记录
```
