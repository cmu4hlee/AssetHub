# AI分析功能配置说明

## 配置AI服务

为了使用真实的AI分析功能，需要在后端配置AI API密钥。系统支持两种AI服务：

1. **阿里百炼（DashScope）** - 推荐，国内访问速度快
2. **OpenAI** - 备选方案

### 方式一：配置阿里百炼智能体应用（推荐）

#### 1. 获取阿里百炼API Key和应用ID

1. 访问 https://dashscope.console.aliyun.com/
2. 登录或注册阿里云账号
3. 开通通义千问服务
4. 创建API密钥
5. 创建智能体应用，获取应用ID

#### 2. 配置环境变量

在 `backend` 目录下创建或编辑 `.env` 文件，添加以下配置：

```env
# 阿里百炼智能体应用配置（推荐）
DASHSCOPE_API_KEY=sk-f0debcc74f684e10a71b379bdf713721
DASHSCOPE_APP_ID=a553a412322e42458604f0ea6fbfb824
DASHSCOPE_BASE_URL=https://dashscope.aliyuncs.com/api/v1/apps
```

**注意**：如果使用默认配置，系统已内置应用ID和API Key，无需额外配置。

### 方式二：配置OpenAI（备选）

#### 1. 获取OpenAI API Key

1. 访问 https://platform.openai.com/api-keys
2. 登录或注册OpenAI账号
3. 创建新的API密钥

#### 2. 配置环境变量

在 `backend` 目录下创建或编辑 `.env` 文件，添加以下配置：

```env
# OpenAI配置（可选，如果使用OpenAI）
OPENAI_API_KEY=your_openai_api_key_here
OPENAI_BASE_URL=https://api.openai.com/v1
OPENAI_MODEL=gpt-3.5-turbo
```

### 3. 重启后端服务

配置完成后，需要重启后端服务使配置生效：

```bash
cd backend
npm start
# 或开发模式
npm run dev
```

## 功能说明

### 资产AI分析功能

系统提供强大的资产AI分析功能，支持多种分析维度：

1. **资产概览分析** 📊 - 分析资产的整体情况、价值分布、使用状态等
2. **资产价值分析** 💰 - 分析资产的价值趋势、折旧情况、投资回报等
3. **资产利用率分析** ⚡ - 分析资产的使用效率、闲置情况、优化建议等
4. **维护成本分析** 🔧 - 分析维护成本、故障率、预防性维护建议等
5. **生命周期分析** 🔄 - 分析资产的生命周期阶段、更新换代建议等
6. **风险评估分析** ⚠️ - 分析资产的风险点、安全隐患、保险建议等
7. **优化建议分析** ✨ - 提供资产配置优化、成本节约、效率提升建议等

### 使用方式

1. **预设维度分析**：选择分析维度，系统自动生成分析提示词
2. **自定义分析**：输入任意分析内容，AI根据内容进行分析
3. **单个资产分析**：针对特定资产进行深度分析
4. **批量资产分析**：对多个资产进行综合分析
5. **分析历史**：查看历史分析记录

### 技术说明

- 系统优先使用阿里百炼智能体应用配置
- 如果未配置阿里百炼，则使用OpenAI配置
- 如果都未配置，系统将使用基础文本分析功能
- 所有分析记录会保存到数据库，方便后续查看

## 注意事项

- API密钥请妥善保管，不要提交到代码仓库
- AI API调用会产生费用，请根据使用量合理配置
- 如果API调用失败，系统会自动回退到基础分析功能
- 阿里百炼使用 `qwen-turbo` 模型，OpenAI使用 `gpt-3.5-turbo` 模型
