## 🚀 验证方法

### 检查配置
```bash
# 检查前端配置
cd Asset/frontend
grep -A 2 "development" vite.config.js

# 检查后端配置
cd ../backend
grep -A 2 "isDevelopment" config/app.config.js
```

### 启动服务验证
```bash
# 启动后端（应该显示端口6001）
cd Asset/backend
npm start

# 启动前端（应该显示端口6000）
cd ../frontend
npm run dev
```

### 检查端口占用
```bash
# 检查开发模式端口
lsof -i:6000
lsof -i:6001

# 检查生产模式端口
lsof -i:4000
lsof -i:4001
```
