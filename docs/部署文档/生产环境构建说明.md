# 生产环境构建说明

**最后更新**: 2026-01-14

---

## 📦 构建步骤

### 1. 前端构建

```bash
cd Asset/frontend
npm run build
```

**构建输出**:
- 输出目录: `frontend/dist/`
- 主要文件:
  - `index.html` - 入口HTML文件
  - `assets/index-*.css` - 样式文件
  - `assets/index-*.js` - JavaScript打包文件

**构建时间**: 约 3-5 秒

### 2. 后端准备

后端是 Node.js 项目，无需构建，但需要：

1. **安装依赖**（如果尚未安装）:
   ```bash
   cd Asset/backend
   npm install
   ```

2. **配置环境变量**:
   - 确保 `.env` 文件存在并配置正确
   - 主要配置项:
     ```bash
     PORT=4001
     DB_HOST=localhost
     DB_PORT=3306
     DB_USER=root
     DB_PASSWORD=your_password
     DB_NAME=zcgl
     ```

## 🚀 启动生产环境

### 方式1: 使用 Vite Preview（推荐用于测试）

```bash
# 前端（80端口）
cd Asset/frontend
npm run preview

# 后端（4001端口）
cd Asset/backend
npm start
```

### 方式2: 使用 Nginx + PM2（推荐用于生产）

#### 前端部署（Nginx）

1. **配置 Nginx**:
   ```nginx
   server {
       listen 80;
       server_name your-domain.com;
       
       root /path/to/Asset/frontend/dist;
       index index.html;
       
       location / {
           try_files $uri $uri/ /index.html;
       }
       
       location /api {
           proxy_pass http://localhost:4001;
           proxy_set_header Host $host;
           proxy_set_header X-Real-IP $remote_addr;
       }
       
       location /uploads {
           proxy_pass http://localhost:4001;
       }
   }
   ```

2. **复制构建文件**:
   ```bash
   sudo cp -r Asset/frontend/dist/* /var/www/html/
   ```

#### 后端部署（PM2）

```bash
# 安装 PM2
npm install -g pm2

# 启动后端服务
cd Asset/backend
pm2 start server.js --name asset-backend

# 设置开机自启
pm2 startup
pm2 save
```

## 📊 构建信息

### 当前构建配置

- **前端端口**: 4000（开发和生产环境统一）
- **后端端口**: 4001（可通过环境变量 `PORT` 配置）
- **构建工具**: Vite 7.3.1
- **打包大小**: 
  - JavaScript: ~1.8 MB (gzip: ~526 KB)
  - CSS: ~8.75 KB (gzip: ~2.62 KB)

### 性能优化建议

⚠️ **注意**: 当前构建输出警告某些 chunk 大于 500KB，建议：

1. **代码分割**:
   ```javascript
   // 使用动态导入
   const Component = lazy(() => import('./Component'));
   ```

2. **手动分块**:
   在 `vite.config.js` 中配置:
   ```javascript
   build: {
     rollupOptions: {
       output: {
         manualChunks: {
           'vendor': ['react', 'react-dom'],
           'antd': ['antd'],
         }
       }
     }
   }
   ```

## 🔧 常见问题

### 1. 构建失败

**问题**: `npm run build` 报错

**解决**:
- 检查 Node.js 版本（建议 v18+）
- 清除缓存: `rm -rf node_modules package-lock.json && npm install`
- 检查 `vite.config.js` 配置是否正确

### 2. 80端口权限问题

**问题**: 无法绑定80端口

**解决**:
- macOS/Linux: 使用 `sudo` 或配置端口转发
- Windows: 通常不需要特殊权限
- 或使用环境变量覆盖: `VITE_PREVIEW_PORT=8080 npm run preview`

### 3. 生产环境API调用失败

**问题**: 前端无法访问后端API

**解决**:
- 检查 Nginx 代理配置
- 检查后端服务是否运行
- 检查 CORS 配置
- 检查防火墙设置
