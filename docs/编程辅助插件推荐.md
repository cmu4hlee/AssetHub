# VSCode/Cursor 编程辅助插件推荐

本文档列出了适合本资产管理系统项目的推荐插件，这些插件可以显著提升开发效率和代码质量。

## 📋 插件列表

### 🔴 必需插件（核心开发工具）

#### 1. **ESLint** (`dbaeumer.vscode-eslint`)
- **作用**: JavaScript/React 代码质量检查
- **为什么需要**: 项目已配置 ESLint，此插件提供实时错误提示和自动修复
- **安装**: 已在 `.vscode/settings.json` 中配置

#### 2. **Prettier** (`esbenp.prettier-vscode`)
- **作用**: 代码自动格式化
- **为什么需要**: 统一代码风格，已在配置中使用
- **安装**: 已在 `.vscode/settings.json` 中配置

---

### 🟡 React 开发增强

#### 3. **ES7+ React/Redux/React-Native snippets** (`dsznajder.es7-react-js-snippets`)
- **作用**: React 代码片段快速生成
- **功能**: 
  - `rafce` → 生成函数组件
  - `useState` → 快速创建状态
  - `useEffect` → 快速创建副作用
- **推荐理由**: 大幅提升 React 开发速度

#### 4. **Auto Rename Tag** (`formulahendry.auto-rename-tag`)
- **作用**: 自动重命名配对的 HTML/JSX 标签
- **示例**: 修改 `<div>` 时，`</div>` 自动同步修改
- **推荐理由**: 避免标签不匹配错误

---

### 🟢 代码智能提示

#### 5. **Path Intellisense** (`christian-kohler.path-intellisense`)
- **作用**: 文件路径自动补全
- **功能**: 输入路径时自动提示项目中的文件
- **推荐理由**: 减少路径输入错误，提升导入文件效率

#### 6. **Error Lens** (`usernamehw.errorlens`)
- **作用**: 在代码行内直接显示错误和警告
- **功能**: 无需悬停即可看到问题
- **推荐理由**: 更直观的错误提示，提升调试效率

---

### 🔵 Git 版本控制增强

#### 7. **GitLens** (`eamodio.gitlens`)
- **作用**: Git 功能增强
- **功能**:
  - 显示每行代码的作者和提交信息

  - 比较不同版本
- **推荐理由**: 强大的 Git 可视化工具

#### 8. **Git History** (`donjayamanne.githistory`)
- **作用**: 查看文件、行和比较的 Git 历史
- **功能**: 图形化显示提交历史

- **功能**:

## 📦 安装方式

### 方式一：通过 VSCode/Cursor 界面安装

1. 打开 VSCode/Cursor
2. 按 `Cmd+Shift+X` (Mac) 或 `Ctrl+Shift+X` (Windows) 打开扩展面板
3. 搜索插件名称或 ID
4. 点击"安装"

### 方式二：一键安装所有推荐插件

1. 打开命令面板：`Cmd+Shift+P` (Mac) 或 `Ctrl+Shift+P` (Windows)
2. 输入 `Extensions: Show Recommended Extensions`
3. 点击"全部安装"

### 方式三：通过命令行安装

```bash
# 安装单个插件（示例）
code --install-extension dbaeumer.vscode-eslint
code --install-extension esbenp.prettier-vscode

# 或使用 Cursor
cursor --install-extension dbaeumer.vscode-eslint
```

## 💡 使用建议

1. **逐步安装**: 不要一次性安装所有插件，先安装高优先级插件，使用一段时间后再添加其他插件
2. **定期更新**: 保持插件更新以获得最新功能和 bug 修复
3. **性能考虑**: 如果编辑器变慢，可以禁用不常用的插件
4. **团队协作**: 建议团队统一使用这些插件，保持开发环境一致

## ❓ 常见问题

**Q: 插件太多会影响性能吗？**
A: 现代编辑器对插件有良好的性能优化，但建议只安装需要的插件。如果遇到性能问题，可以禁用不常用的插件。

**Q: 这些插件都是免费的吗？**
A: 本文档推荐的所有插件都是免费的，部分插件有付费的高级版本（可选）。

**Q: 如何卸载插件？**
A: 在扩展面板中找到插件，点击"卸载"即可。
