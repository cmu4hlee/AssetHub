# 杯架伸缩立柱 - 混合材料版

## 产品描述

一个使用 **PVC 管 + 3D 打印连接件** 的混合材料伸缩立柱。底座插入杯架，管子向上延伸顶住车顶天窗，可放置手机、平板或其他设备。

---

## 设计理念

| 方案 | 优点 | 缺点 |
|------|------|------|
| **纯 3D 打印** | 一体成型 | 800mm 超出大多数打印机范围 |
| **混合材料版** ✓ | 强度高、成本低、易获取 | 需要购买管材 |
| 铝合金版 | 轻便坚固 | 加工螺纹困难 |

## 组装步骤

### 第一步：准备管材

```
1. 切割 PVC DN40 管: 700mm
2. 切割 PVC DN32 管: 150mm
3. 用砂纸打磨切口，去除毛刺
4. 确保管子切口平整垂直
```

### 第二步：安装底座

```
1. 将 PVC DN40 管插入底座 (深度约 25mm)
2. 装上底座内衬套
3. 旋紧 3 颗 M6 顶丝固定
```

### 第三步：安装外管夹紧环

```
1. 将外管夹紧环套在 PVC DN40 管上
2. 移动到距离底座约 20mm 的位置
3. 旋紧 3 颗 M6 顶丝固定
```

### 第四步：安装内管

```
1. 将 PVC DN32 管插入 DN40 管内 (深度约 30mm)
2. 调整伸出长度 (建议 80-100mm)
3. 装上内管夹紧环
4. 旋紧 3 颗 M6 顶丝固定
```

### 第五步：安装顶板

```
1. 将 PVC DN32 管插入顶部连接器
2. 旋紧 4 颗 M4 沉头螺丝固定
3. 在顶部贴上橡胶防滑垫
```

### 第六步：测试安装

```
1. 将底座放入杯架
2. 旋转内管调节高度
3. 确保立柱稳固顶住车顶
4. 旋紧所有顶丝
```

## 高度计算

```
杯架深度: 通常 40-60mm
底座高度: 35mm
外管长度: 700mm
内管伸出: 0-100mm
顶板高度: 32mm

总高度 = 35 + 700 + (0~100) + 32 = 767 ~ 867mm
```

## 使用说明

### 调整高度
1. 松开内管夹紧环的 3 颗顶丝
2. 上下滑动内管调整高度
3. 旋紧顶丝固定

### 安装设备
- 直接放置在顶板橡胶垫上
- 可用手机夹配件夹持手机
- 可用球头连接器多角度调节

### 注意事项
- 行驶前务必确认立柱已稳固
- 紧急刹车时可能移位，建议辅助固定
- 定期检查顶丝是否松动
- 避免在颠簸路面高速行驶

## 常见问题

**Q: 管子插入底座后太松怎么办？**
A: 可以在管子外缠几圈绝缘胶带或生料带

**Q: 如何防止立柱在杯架里转动？**
A: 底座内部有定位扁位设计，配合顶丝固定

**Q: 800mm 太长/太短？**
A: 修改 `OUTER_PIPE_LENGTH` 参数重新生成模型
