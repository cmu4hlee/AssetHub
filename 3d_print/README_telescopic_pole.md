# 杯架伸缩立柱 - 天窗支撑架

## 产品描述

一个可调节的螺纹伸缩立柱，底部适配汽车杯架，顶部可延伸顶住车顶天窗位置，用于固定手机、平板或其他设备。

---

## 设计特点

| 特点 | 说明 |
|------|------|
| **螺纹伸缩** | 双层管结构，通过旋转调节高度 |
| **杯架适配** | 底部直径 70mm，适配大多数圆形杯架 |
| **模块化设计** | 底座、外管、内管、顶板分体打印 |
| **多种顶配** | 可选配吸盘、手机夹、球头连接件 |

## 使用说明

### 组装步骤

1. **检查零件**
   - 确保所有打印件无毛刺
   - 测试螺纹配合（需要少量打印测试）

2. **组装螺纹**
   - 将内管旋入外管
   - 调整到合适高度
   - 用胶水或螺纹锁固剂固定外管与底座

3. **安装顶板**
   - 将顶板连接到内管顶部
   - 可使用螺纹或胶水固定

4. **放入杯架**
   - 将底座插入杯架
   - 向下压实确保稳固

5. **调节高度**
   - 旋转内管/外管调整高度
   - 顶住天窗顶部

### 高度计算

```
总高度 = 底座高度 + 外管长度 + 内管伸出长度 + 顶板高度
       = 25 + 80 + (0~70) + 10
       = 115 ~ 185mm
```

### 安全提示
- 行驶前务必确认立柱已稳固顶住
- 紧急刹车时可能脱落，需谨慎使用
- 建议额外使用魔术贴或夹子辅助固定
- 吸盘版本需定期检查吸附力

## 改装建议

### 增强稳定性
1. 底座填充沙袋或配重块
2. 使用橡胶垫增加摩擦力
3. 缠绕防滑胶带

### 增加功能性
1. 在顶板预留螺丝孔安装通用支架
2. 侧面开槽安装磁铁
3. 添加手机无线充电线圈安装位

## 参数调整指南

在代码中修改以下参数适配你的车型：

```openscad
BASE_OUTER_DIAMETER = 70;     // 底座直径 (根据杯架调整)
BASE_HEIGHT = 25;             // 底座高度
OUTER_TUBE_LENGTH = 80;       // 外管长度 (可加长)
MAX_EXTENSION = 150;          // 最大伸展量
TOP_PLATE_DIAMETER = 60;      // 顶板直径
```
