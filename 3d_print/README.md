# 车载手机支架设计说明

## 产品规格

### 适配尺寸
- **适用手机**: 大屏手机 (6.1-6.9寸)
- **手机宽度范围**: 约 75-78mm
- **手机厚度**: 约 10mm (含手机壳)

### 安装方式
- **出风口夹持式** - 夹在空调出风口叶片上

### 主要尺寸
| 部位 | 尺寸 |
|------|------|
| 手机托盘深度 | 50mm |
| 边缘高度 | 15mm |
| 夹爪宽度 | 40mm |
| 夹爪高度 | 25mm |
| 颈部长度 | 60mm |
| 总高度 | 约 140mm |

---

## 零件组成

### 一体化设计
本设计为**一体化打印**，无需组装。包含以下功能区域：

1. **出风口夹持部**
   - 带防滑齿设计，抓附力强
   - 可调节间隙适应不同出风口
   - 减重孔设计减少材料使用

2. **球形关节颈部**
   - 可 360° 旋转调节角度
   - 稳固的球窝关节结构

3. **手机托盘**
   - U 型设计固定手机底部
   - 两侧护栏防止侧滑
   - 底部通风口不影响空调效果

## 参数调整指南

在 `car_phone_mount.scad` 文件中可调整以下参数：

```openscad
PHONE_WIDTH = 78;        // 手机宽度 (mm)
MOUNT_DEPTH = 50;        // 托盘深度 (mm)
VENT_GAP = 18;           // 风口间隙 (mm)
CLAMP_WIDTH = 40;        // 夹爪宽度 (mm)
```

### 不同手机的参数建议
| 手机类型 | 示例手机 | PHONE_WIDTH |
|---------|---------|-------------|
| 小屏手机 | iPhone SE | 67 |
| 标准手机 | iPhone 15/16 | 71 |
| 大屏手机 | iPhone Pro Max | 78 |
| 安卓大屏 | 小米 14 Ultra | 75 |

## 打印设置

### 推荐参数
| 设置项 | 推荐值 |
|--------|--------|
| 层高 | 0.2mm |
| 填充率 | 30-50% |
| 壁厚 | 2-3 层 |
| 打印材料 | **PETG (推荐)** 或 PLA |
| 喷嘴温度 | PLA: 200-215°C / PETG: 235-245°C |
| 热床温度 | PLA: 60°C / PETG: 80°C |
| 打印速度 | 40-60mm/s |

### 打印方向
- **推荐**: 夹持部朝下，托盘朝上
- 需要支撑结构（主要在托盘底部和颈部连接处）
- 支撑类型: Tree 或 Lines 支撑

---

## 打印后处理

1. **去除支撑**
   - 小心去除支撑结构
   - 可用砂纸打磨支撑痕迹

2. **表面处理**
   - 细砂纸 (320-600目) 打磨
   - 如需更光滑可进行抛光处理

3. **增强防滑**
   - 夹持部可缠绕硅胶防滑条
   - 托盘底部可粘贴 3M 胶垫

4. **可选升级**
   - 在球窝关节处可涂抹少量润滑油增强灵活性
   - 添加无线充电线圈兼容位（需自行设计）

---

## 使用说明

1. 将夹持部插入出风口叶片间
2. 调节间隙旋钮（可通过 3D 打印微调件调整）
3. 放置手机到底托上
4. 旋转颈部调整最佳观看角度
