# 《医学装备整体运维管理服务规范》功能完善实施报告

## 一、实施概述

根据《医学装备整体运维管理服务规范》（2025.12.03 送批稿）的要求，对 AssetHub 系统进行了功能完善，主要补充了以下核心功能模块。

---

## 二、已完成的功能模块

### ✅ 1. 分级保养体系

#### 数据库表
- `asset_risk_levels` - 设备风险等级评估表
- `maintenance_level_templates` - 分级保养模板表
- `maintenance_level_plans` - 分级保养计划表

#### API 接口
| 接口 | 说明 |
|------|------|
| `GET /api/compliance/maintenance-level/templates` | 获取保养模板列表 |
| `POST /api/compliance/maintenance-level/templates` | 创建保养模板 |
| `PUT /api/compliance/maintenance-level/templates/:id` | 更新保养模板 |
| `DELETE /api/compliance/maintenance-level/templates/:id` | 删除保养模板 |
| `GET /api/compliance/maintenance-level/plans` | 获取保养计划列表 |
| `POST /api/compliance/maintenance-level/plans/generate` | 自动生成保养计划 |
| `POST /api/compliance/maintenance-level/plans/:id/execute` | 执行保养计划 |
| `GET /api/compliance/maintenance-level/risk-levels` | 获取风险等级列表 |
| `POST /api/compliance/maintenance-level/risk-levels` | 评估设备风险等级 |
| `GET /api/compliance/maintenance-level/statistics` | 保养统计 |

#### 功能特性
- ✅ 支持日常保养、一级、二级、三级保养分级体系
- ✅ 设备风险等级评估（高/中/低）
- ✅ 自动根据风险等级生成保养计划
- ✅ 保养模板管理
- ✅ 保养执行记录和审核

---

### ✅ 2. 开机率统计

#### 数据库表
- `asset_operation_logs` - 设备运行记录表
- `asset_uptime_statistics` - 设备开机率统计表

#### API 接口
| 接口 | 说明 |
|------|------|
| `POST /api/compliance/uptime/operation-logs` | 记录设备运行数据 |
| `GET /api/compliance/uptime/operation-logs` | 获取运行记录 |
| `POST /api/compliance/uptime/calculate` | 计算开机率统计 |
| `GET /api/compliance/uptime/statistics` | 获取开机率统计 |
| `GET /api/compliance/uptime/overview` | 获取开机率概览 |
| `POST /api/compliance/uptime/batch-operation-logs` | 批量录入运行数据 |

#### 功能特性
- ✅ 设备运行时间记录
- ✅ 计划/实际运行时长对比
- ✅ 停机原因分类（保养/维修/故障/检查/其他）
- ✅ 按月自动统计开机率
- ✅ 设备分类统计（生命支持类/大型设备/常规设备）
- ✅ 部门开机率排名
- ✅ 低开机率设备预警

## 四、API 路由汇总

所有合规性功能路由统一前缀：`/api/compliance`

```
/api/compliance
├── /maintenance-level        # 分级保养
│   ├── /templates           # 保养模板
│   ├── /plans               # 保养计划
│   ├── /risk-levels         # 风险等级
│   └── /statistics          # 统计
├── /uptime                   # 开机率统计
│   ├── /operation-logs      # 运行记录
│   ├── /calculate           # 计算统计
│   ├── /statistics          # 统计数据
│   ├── /overview            # 概览
│   └── /batch-operation-logs # 批量录入
├── /staff                    # 人员资质
│   ├── /qualifications      # 资质管理
│   └── /training-records    # 培训记录
├── /special-equipment        # 特种设备
│   ├── /                    # 设备管理
│   ├── /inspections         # 检验记录
│   └── /statistics          # 统计
└── /safety-inspection        # 安全检测
    ├── /                    # 检测管理
    └── /statistics          # 统计
```
